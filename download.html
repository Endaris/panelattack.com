Panel Attack is very easy to set up. On Windows, just download the zip file above, unzip it, and run panel.exe. Set your controls, set your name, and you're ready to go!

Setting up Panel Attack on Mac

Download and install love 11.5 for Mac from https://love2d.org.
Download panel.zip, unzip it, and change the file extension of panel.exe from .exe to .love. After that you can start the game by doubleclicking panel.love.

Setting up Panel Attack on Linux

Download the AppImage from https://love2d.org.
Right-click the file and select Properties -> Permissions. Check the option that allows the file to run as a program.
Download panel.zip, unzip it, and change the file extension of panel.exe from .exe to .love.
After that you can start the game by dragging and dropping panel.love on top of the AppImage or start the AppImage via the terminal and pass panel.love as the argument.

Setting up Panel Attack on Android
